# Backendprojekt fuer echte Dinos <3 

## Projektbeschreibung
Dieses Projekt ist ein Backendprojekt für echte Dinos. Es ist ein einfaches REST-API, welches die CRUD-Operationen für Dinos ermöglicht.

## Technologien
- Java
- Spring Boot

## Endpunkte
- GET /dinos
- GET /dinos/{id}
- POST /dinos
- PUT /dinos/{id}
- DELETE /dinos/{id}

## Datenmodell
```json
{
  "id": 1,
  "species": "T-Rex",
  "alter": 100,
  "gewicht": 1000.0
}
```

## Setup
1. Projekt klonen
2. Projekt in IDE öffnen
3. Projekt starten
4. API testen
5. Fertig!
